== 120 Install Git

Git is required for checking out projects from version control.

  sudo yum -y install zlib-devel openssl-devel cpio expat-devel gettext-devel

Install from source

  cd ~/code/source
  wget http://git-core.googlecode.com/files/git-1.7.9.tar.gz
  tar xvzf git-1.7.9.tar.gz
  rm git-1.7.9.tar.gz
  cd git-*
  ./configure
  make
  sudo make install

Verify Git version

  git --version

  <i>git version 1.7.9</i>
