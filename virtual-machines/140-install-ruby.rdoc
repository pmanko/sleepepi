== 140 Install Ruby


=== 141 Install Ruby using RVM

  rvm install 1.9.3-p194

Set default Ruby

  rvm alias create default ruby-1.9.3-p194

Activate Ruby 1.9.3-p194

  rvm 1.9.3-p194

Verify Ruby version

  ruby -v

  ruby 1.9.3p194 (2012-04-20 revision 35410) [x86_64-linux]

=== 142 Update Ruby Gems

  gem update --system
